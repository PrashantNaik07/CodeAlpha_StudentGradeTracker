# 🎓 Student Grade Tracker (Java Console Application)

## 📌 Project Overview
The **Student Grade Tracker** is a console-based Java application designed to manage student academic records efficiently. It allows users to add, update, delete, and search student details while automatically calculating performance metrics such as average marks, grade, class report, and rank list.

This project is built using **Core Java and Object-Oriented Programming (OOP)** concepts.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add new student with ID and name
- View all students in tabular format
- Search student by ID
- Update marks for specific subjects
- Delete student records

---

### 📊 Academic Analysis
- Automatic average calculation
- Grade system:
  - A+ (90+)
  - A (80–89)
  - B (70–79)
  - C (60–69)
  - D (50–59)
  - F (<50)

---

### 🏆 Advanced Features
- Rank list based on average marks
- Class report including:
  - Class average
  - Highest score
  - Lowest score
- Subject-wise marks display
